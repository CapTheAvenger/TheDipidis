📖 How to Use This Website

Complete Guide to Pokemon TCG Analysis & Deck Building

🎯 Website Overview

This website provides comprehensive Pokemon TCG meta analysis, deck building tools, and tournament data. Access Japanese City League results, international tournament data, and detailed card statistics to build competitive decks.

📂 Available Tabs

🇯🇵 City League Meta

Purpose: View meta development and archetype stats from Japanese City League tournaments

What you see: Tournament date, archetype usage, placement statistics, meta share graphs

Best for: Understanding the current Japanese meta trends and archetype popularity over time

📊 City League Deck Analysis

Purpose: Analyze specific decks from Japanese City Leagues and build your own

Features:

  • 🗓️ Date Filter: Filter tournaments by date range (From/To dates)
  • 🔍 Deck Search: Type to find specific archetypes
  • 🎯 Deck Selector: Choose from available decks
  • 🦂 Card Filter: Show core cards (>90%), common (>70%), or all cards
  • 📊 Statistics: View deck size, usage, average placement
  • 🎴 Card Overview: See all cards with usage%, rarity options, type filters
  • 🛠️ Deck Builder: Auto-generate or manually build decks, copy to clipboard
  • 🔄 Deck Compare: Compare two decklists and see differences with visual badges

🎮 Current Meta

Purpose: Compare Limitless Online data with major tournament results

What you see: Side-by-side archetype comparisons, win rates, meta share

Best for: Understanding differences between online ladder and real tournaments

📈 Current Meta Deck Analysis

Purpose: Analyze current meta decks from Limitless Online and tournaments

Special Features:

  • 🎮 Format Filter: View All / Limitless Decks only / Tournament Decks only
  • 📊 Win Rate Stats: Total online tournament win rate
  • ⚔️ Matchup Analysis: See best and worst matchups with win rates and game records
  • 🔍 Opponent Search: Search specific matchups to analyze head-to-head data

🏆 Past Meta

Purpose: Analyze historical tournament winners and past format decklists

Features: Format filter, tournament filter, deck analysis just like other tabs

Best for: Learning from winning decks, understanding format evolution

🧰 Cards

Purpose: Browse all available Pokemon TCG cards

Features:

  • 🔍 Search: Find cards by name
  • 🎴 Type Filter: Filter by Pokemon, Supporter, Item, Tool, Stadium, Energy, Ace Spec
  • 🎯 Set Filter: Select specific card sets
  • 💎 Rarity Filter: Filter by card rarity
  • 📊 Sort: Organize cards by different criteria
  • 🖼️ View Mode: Grid view with card images

🛠️ Using the Deck Builder

📋 Quick Start

  1. Select a Deck: Use the deck selector dropdown to choose an archetype
  2. View Cards: See Card Overview with all cards and their usage statistics
  3. Auto-Generate: Click "Generate Low Rarity Deck" or "Generate Max Rarity Deck" for instant 60-card deck
  4. Manual Building: Click on individual cards to add them (left click +1, right click +4)
  5. Copy Deck: Click "Copy Deck" to get the complete decklist in Pokemon TCG Live format

🎨 Card Display Modes

⬇️ Low Rarity: Shows cheapest version of each card (budget-friendly)

⬆️ Max Rarity: Shows highest rarity version (full art, secret rare, etc.)

🌍 All Prints: Shows all available versions of each card

🖼️ View Modes

Table View: Detailed list with card names, sets, usage%, and controls

Grid View: Visual card display with images for quick overview

🔄 Deck Compare Feature

Compare your current deck with a previous version:

  1. Build or generate a deck in "Your Deck" section
  2. Click "🔄 Compare Deck" button
  3. Paste your old decklist in Pokemon TCG Live format
  4. Click "Compare"
  5. See color-coded badges:
    • 🟡 Gold = New cards added
    • 🟢 Green = Count increased
    • 🔴 Red = Count decreased or removed
    • ⚫ Gray = Unchanged
💡 Tips:
  • Deck Compare supports international prints (e.g., MEG 76 matches ASC 112 for Riolu)
  • Basic Energies from different sets are recognized as the same card
  • Japanese set codes (M3, M2a, etc.) are fully supported

💡 Tips & Tricks

  • 🎯 Use Date Filters: Focus on recent tournaments for current meta insights
  • 📊 Check Usage %: Cards with >90% usage are core to the archetype
  • 🔍 Search is Your Friend: Use search boxes to quickly find specific cards or decks
  • 🌍 International Prints: The website recognizes international reprints automatically
  • 💰 Budget Building: Use "Low Rarity" option for affordable deck versions
  • 🎨 Grid View: Perfect for visual deck overview and spotting missing cards
  • 📋 Copy Function: All deck copies use Pokemon TCG Live format for easy import
  • 🔄 Compare Decks: Great for tracking deck evolution and understanding changes

❓ Frequently Asked Questions

Q: How often is the data updated?

A: Check the "Last Update" timestamp in the footer. Data is updated regularly when new tournaments occur.

Q: Can I use this for deck building in Pokemon TCG Live?

A: Yes! All "Copy Deck" functions output decks in Pokemon TCG Live format. Just paste directly into the game.

Q: What does "international prints" mean?

A: The same card printed in different sets (e.g., Riolu MEG 76 = Riolu ASC 112). The website recognizes these automatically.

Q: Why are Japanese cards (M3, M2a) shown?

A: Japanese sets often release months before international versions. These give early insight into upcoming cards and strategies.

Q: How do I add cards to my deck manually?

A: In the Card Overview section, left-click a card to add 1 copy, or right-click to add 4 copies. Cards appear in "Your Deck" section below.

Q: What's the difference between "Low Rarity" and "Max Rarity"?

A: Low Rarity shows the cheapest printings (budget decks), Max Rarity shows premium versions (full arts, secret rares). Gameplay is identical.
