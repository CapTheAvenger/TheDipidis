📖 How to Use This Website

Complete Guide to Pokemon TCG Analysis, Deck Building & Playtester — v45 (March 2026)

🎯 Website Overview

This website provides comprehensive Pokémon TCG meta analysis, deck building tools, a full card database, a collection tracker, and a fully-featured playtest simulator. Access Japanese City League results, international tournament data, and detailed card statistics to build competitive decks — and test them directly in your browser.

🆕 New in v45: Card Database Reliability — filter panel wiring was rebuilt; all filter groups now populate correctly and loading has a safe fallback if English set mapping cannot be resolved. Proxy Queue Fix — removed duplicate proxy queue container; proxy rendering is reliable again. Multiplayer Playtester Script — firebase-multiplayer.js now loads before playtester so online multiplayer actions are available. Rarity / Category / Deck Coverage Filters — were missing; these panels are now auto-generated at load time.

🛠️ Deck Building — Full Guide

📋 Quick Start

  1. Select a Deck from the dropdown
  2. View Cards in the Card Overview with usage statistics
  3. Auto-Generate: Click "🎯 Auto Generate (Max Consistency)" for a tournament-tested 60-card deck
  4. Manual Building: Left-click a card to add 1 copy, right-click to add 4
  5. Meta Analysis: Load "Meta Card Analysis" to see the most played cards across the Top 10 archetypes
  6. Copy Deck: Click "Copy Deck" for Pokémon TCG Live format

🎯 Max Consistency Algorithm (v18)

The auto-generate algorithm was completely rewritten in v18 for better accuracy:

  • 5% meta-share cutoff: Cards appearing in <5% of decks are ignored (no false positives)
  • Smart rounding: Uses avgCountWhenUsed (average among decks that actually include the card, not all decks) — so e.g. 3.07× → rounds to 3, not 4
  • Combined Variants (NEW): Cards from different sets (e.g. Riolu MEG 76 = Riolu ASC 112) are automatically combined — usage% reflects the true card popularity regardless of which print was used
  • Rare Candy guarantee: Stage 2 evolution lines always include Rare Candy
  • Ace Spec slot: 1 Ace Spec added automatically based on data
  • Fallback fill: Remaining slots filled with highest-priority cards to always reach 60

⬆️ Max Rarity: Highest rarity version (full art, secret rare, etc.)

⭐ Rarity Switcher (★)

Every card in Card Overview, Deck Builder grid, and My Decks has a yellow ★ button. Click it to open a modal showing all available prints of that card — regular, reverse holo, full art, trainer gallery, promo, etc.

  • Click any version to immediately replace the print in the active deck entry
  • Each switcher card shows exact owned count for that print
  • ✨ Andere Prints line appears if you own additional international prints of the same card

🔄 Deck Compare Feature

Compare your current deck with any saved deck or previous version:

  1. Build or generate a deck in "Your Deck" section
  2. Click "🔀 Compare Deck"
  3. Paste a list in Limitless/PTCGL format or choose a saved deck via search
  4. Select compare mode: Functional (prints merged) or Exact print
  5. Color-coded badges: Gold = Added   Red = Removed   Green = Unchanged

❓ Frequently Asked Questions

Q: How often is the data updated?

A: Check the "Last Update" timestamp in the footer. Data is updated regularly as new tournaments occur.

Q: Can I use this for deck building in Pokémon TCG Live?

A: Yes! All "Copy Deck" functions output decks in Pokémon TCG Live format. Paste directly into the game.

Q: What does "international prints" mean?

A: The same card printed in different sets (e.g., Riolu MEG 76 = Riolu ASC 112). The website recognizes these automatically across all tabs.

Q: Why are Japanese cards (M3, M2a) shown?

A: Japanese sets release months before international versions. They give early insight into upcoming cards and strategies.

Q: My Rarity Switcher showed a "?" instead of ★ before — is that fixed?

A: Yes, fixed in v36. All broken symbol characters (Rarity Switcher ★, owned ✓ badge, Ace Spec ★ column) are now correctly displayed across every tab.

Q: The Playtester loaded wrong card arts from My Decks — is that fixed?

A: Yes, fixed in v35. The new direct lookup bypasses the string roundtrip, so cards with Trainer Gallery numbers (TG01), Gallery Gold (GG01), full arts, and promos all resolve to the exact saved print.

Q: What are "Combined Variants"?

A: New in v37. When the same card is printed in multiple sets (e.g. Riolu in MEG and ASC), the website automatically combines their statistics. Usage%, average count, and deck inclusion are calculated across all prints. The Deck Builder, Auto-Complete, and Card Overview all respect this — so you see the true popularity of a card regardless of which set version players used.

Q: Why do Special Energies appear before Basic Energies?

A: This is the correct sort order (fixed in v37). Special Energies (Jet Energy, Double Turbo, etc.) sort before Basic Energies (Fire, Water, etc.) across all views — matching standard deck list conventions.

Q: What is "Max Consistency" Auto-Generate?

A: The v18 algorithm looks at every deck using a card, calculates the average count among those decks (not all decks), then rounds to the nearest integer. A 5% meta-share cutoff prevents fringe cards from being included. The result is a tournament-realistic 60-card deck.

Q: Do my decks and collection sync between desktop and mobile?

A: Yes, as long as you're signed in with the same account. Firebase syncs everything in real-time.

💾 Meine Sammlung & Wunschliste

Behalte den Überblick über Karten, die du besitzt und kaufen möchtest. Beide Bereiche sind durchsuchbar und mit deinem Konto synchronisiert.

🔍 Suchfunktionen:

  • Suche nach Kartenname, Set-Code oder Kartennummer
  • Echtzeit-Filterung — Ergebniszähler zeigt "X von Y Karten"

💰 Karten-Aktionen:

  • Cardmarket-Preis anzeigen — klicke den Preis-Button, um auf Cardmarket zu öffnen

  • Rotes × zum Entfernen aus Sammlung / Wunschliste

🎮 Meine Decks

Speichere Decks aus jedem Deck Builder Tab. In Firebase gespeichert — auf allen Geräten verfügbar.

💾 Decks speichern:

  • Klicke "💾 In Meine Decks speichern" in jedem Deck Builder
  • Benenne dein Deck (Archetyp-Name ist vorausgefüllt)
  • Exakte Druckversionen werden gespeichert (Set-Code + Nummer)
  • Über alle angemeldeten Geräte synchronisiert

⚔️ Playtest von Meine Decks (NEU):

  • Klicke den ⚔️ Playtest-Button in der Meine-Decks-Kopfzeile

  • Klicke 🚀 Playtest starten — beide Decks laden mit exakten gespeicherten Prints (TG01, GG01, Full Arts funktionieren alle korrekt)

⭐ Karten-Aktions-Buttons auf jeder Karte:

  • ★ Seltenheit/Print (gelb): Zu einem anderen Print derselben Karte wechseln
  • 💰 Preis (orange/grau): Aktueller Cardmarket-Preis — klicken, um Angebot zu öffnen
  • ✓ Sammlung (grün): Als besessen / nicht besessen markieren
  • ❤️ Wunschliste (orange): Zur Wunschliste hinzufügen / entfernen

🧮 Sammlungs-Badge-Logik (exakter Print):

  • Badge oben links zeigt, wie viele Kopien du von diesem exakten Print besitzt (Name + Set + Nummer)
  • 🟢 Grün = genug Kopien für Deck-Anzahl, 🟠 Orange = teilweise vorhanden, kein Badge = 0 exakte Kopien
  • ✨ Shiny-Badge: erscheint, wenn du andere internationale Prints derselben Karte besitzt

🔄 Deck-Vergleich:

  • Klicke ⚖️ auf einer gespeicherten Deck-Karte, um Vergleichsquelle zu öffnen
  • Quelle wählen: Limitless/PTCGL Einfügen oder Gespeicherte Deck-Suche
  • Einfügemodus zeigt eine Live-Parser-Vorschau vor dem Vergleich
  • Vergleichs-Modal unterstützt Funktional (Prints zusammengeführt) vs. Exakter Print

🛠️ Deckbau — Vollständige Anleitung

📋 Schnellstart

  1. Wähle ein Deck aus dem Dropdown
  2. Karten ansehen in der Kartenübersicht mit Nutzungsstatistiken
  3. Auto-Generieren: Klicke "🎯 Auto-Generieren (Max. Konsistenz)" für ein turniergetestetes 60-Karten-Deck
  4. Manueller Bau: Linksklick auf eine Karte fügt 1 Kopie hinzu, Rechtsklick fügt 4 hinzu
  5. Meta-Analyse: "Meta Karten-Analyse" laden, um die meistgespielten Karten in den Top 10 Archetypen zu sehen
  6. Deck kopieren: "Deck kopieren" für Pokémon TCG Live Format klicken

🎯 Max-Konsistenz-Algorithmus (v18)

Der Auto-Generierungs-Algorithmus wurde in v18 komplett überarbeitet für bessere Genauigkeit:

  • 5% Meta-Share-Grenzwert: Karten, die in <5% der Decks vorkommen, werden ignoriert (keine Fehlalarme)
  • Intelligentes Runden: Verwendet avgCountWhenUsed (Durchschnitt unter Decks, die die Karte tatsächlich spielen, nicht alle Decks) — z.B. 3,07× → wird auf 3 gerundet, nicht 4
  • Kombinierte Varianten (NEU): Karten aus verschiedenen Sets (z.B. Riolu MEG 76 = Riolu ASC 112) werden automatisch kombiniert — Nutzungs-% spiegelt die wahre Kartenpopularität wider, unabhängig davon, welcher Print verwendet wurde
  • Rare Candy Garantie: Stufe-2-Entwicklungslinien enthalten immer Rare Candy
  • Ace Spec Slot: 1 Ace Spec wird automatisch basierend auf Daten hinzugefügt
  • Fallback-Auffüllung: Verbleibende Slots werden mit Karten höchster Priorität gefüllt, um immer 60 zu erreichen

❓ Häufig gestellte Fragen

F: Wie oft werden die Daten aktualisiert?

A: Prüfe den "Letzte Aktualisierung"-Zeitstempel im Footer. Daten werden regelmäßig aktualisiert, wenn neue Turniere stattfinden.

F: Kann ich das zum Deckbau in Pokémon TCG Live verwenden?

A: Ja! Alle "Deck kopieren"-Funktionen geben Decks im Pokémon TCG Live Format aus. Direkt ins Spiel einfügen.

F: Was bedeutet "internationale Prints"?

A: Dieselbe Karte, die in verschiedenen Sets gedruckt wird (z.B. Riolu MEG 76 = Riolu ASC 112). Die Website erkennt diese automatisch über alle Tabs hinweg.

F: Warum werden japanische Karten (M3, M2a) angezeigt?

A: Japanische Sets erscheinen Monate vor den internationalen Versionen. Sie geben frühe Einblicke in kommende Karten und Strategien.

F: Mein Seltenheits-Wechsler zeigte vorher ein "?" statt ★ — ist das behoben?

A: Ja, behoben in v36. Alle defekten Symbolzeichen (Seltenheits-Wechsler ★, Besitz-✓-Badge, Ace Spec ★-Spalte) werden jetzt korrekt in jedem Tab angezeigt.

F: Der Playtester hat falsche Karten-Arts aus Meine Decks geladen — ist das behoben?

A: Ja, behoben in v35. Der neue direkte Lookup umgeht den String-Roundtrip, sodass Karten mit Trainer Gallery Nummern (TG01), Gallery Gold (GG01), Full Arts und Promos alle zum exakten gespeicherten Print aufgelöst werden.

F: Was sind "Kombinierte Varianten"?

A: Neu in v37. Wenn dieselbe Karte in mehreren Sets gedruckt wird (z.B. Riolu in MEG und ASC), kombiniert die Website automatisch ihre Statistiken. Nutzungs-%, Durchschnittszahl und Deck-Einbindung werden über alle Prints hinweg berechnet. Der Deck Builder, Auto-Complete und die Kartenübersicht respektieren dies — sodass du die wahre Popularität einer Karte siehst, unabhängig davon, welche Set-Version die Spieler verwendet haben.

F: Warum erscheinen Spezial-Energien vor Basis-Energien?

A: Das ist die korrekte Sortierreihenfolge (behoben in v37). Spezial-Energien (Jet Energy, Double Turbo usw.) sortieren vor Basis-Energien (Feuer, Wasser usw.) in allen Ansichten — passend zu Standard-Decklisten-Konventionen.

F: Was ist "Max. Konsistenz" Auto-Generieren?

A: Der v18-Algorithmus betrachtet jedes Deck, das eine Karte nutzt, berechnet die durchschnittliche Anzahl unter diesen Decks (nicht allen Decks), und rundet dann auf die nächste ganze Zahl. Ein 5%-Meta-Share-Grenzwert verhindert, dass Randkarten einbezogen werden. Das Ergebnis ist ein turnierrealistisches 60-Karten-Deck.

F: Werden meine Decks und Sammlung zwischen Desktop und Handy synchronisiert?

A: Ja, solange du mit demselben Konto angemeldet bist. Firebase synchronisiert alles in Echtzeit.
