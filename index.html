📖 How to Use This Website

Complete Guide to Pokemon TCG Analysis, Deck Building & Playtester — v38 (March 2026)

🎯 Website Overview

This website provides comprehensive Pokémon TCG meta analysis, deck building tools, a full card database, a collection tracker, and a fully-featured playtest simulator. Access Japanese City League results, international tournament data, and detailed card statistics to build competitive decks — and test them directly in your browser.

🆕 New in v38: Professor Oak Help Button — every tab now has a Professor Oak icon that opens context-specific help. Auto-Save Decks — your deck builder state is saved to localStorage and restored on page reload. Proxy Print Layout — @media print CSS ensures standard 6.3cm × 8.8cm card sizes for physical sleeve-fit proxies. TCG Rule Enforcement — Judge now draws exactly 4 cards, Iono draws equal to remaining prizes (per official rules). Deck Compare Fix — same cards from different prints (e.g. Boss's Orders PAL vs BRS) no longer show as false differences. Card Database Sort — sets now sort newest→oldest with unique order values for all 150+ sets.

🛠️ Deck Building — Full Guide

📋 Quick Start

  1. Select a Deck from the dropdown
  2. View Cards in the Card Overview with usage statistics
  3. Auto-Generate: Click "🎯 Auto Generate (Max Consistency)" for a tournament-tested 60-card deck
  4. Manual Building: Left-click a card to add 1 copy, right-click to add 4
  5. Meta Analysis: Load "Meta Card Analysis" to see the most played cards across the Top 10 archetypes
  6. Copy Deck: Click "Copy Deck" for Pokémon TCG Live format

🎯 Max Consistency Algorithm (v18)

The auto-generate algorithm was completely rewritten in v18 for better accuracy:

  • 5% meta-share cutoff: Cards appearing in <5% of decks are ignored (no false positives)
  • Smart rounding: Uses avgCountWhenUsed (average among decks that actually include the card, not all decks) — so e.g. 3.07× → rounds to 3, not 4
  • Combined Variants (NEW): Cards from different sets (e.g. Riolu MEG 76 = Riolu ASC 112) are automatically combined — usage% reflects the true card popularity regardless of which print was used
  • Rare Candy guarantee: Stage 2 evolution lines always include Rare Candy
  • Ace Spec slot: 1 Ace Spec added automatically based on data
  • Fallback fill: Remaining slots filled with highest-priority cards to always reach 60

⚔️ Playtester — Complete Guide

The Playtester Sandbox simulates a real 2-player Pokémon TCG game in your browser. Load any two decklists and goldfishing practice, test matchups, or study lines.

Loading Decks

Method 1 — Paste & Load

Go to ⚔️ Playtester Sandbox tab → paste a Pokémon TCG Live decklist into the P1 and P2 boxes → click 📥 Load Deck → click Start. Exact prints are resolved by set+number (TG01, GG01, promo numbers all work).

Method 2 — From Deck Builder (any tab)

Click the 🃏 Playtest button in any Deck Builder. A setup modal appears — paste the opponent's deck or skip (mirror match). Click Start.

Method 3 — From My Decks (NEW)

Go to Profile → My Decks → click ⚔️ Playtest header button → pick Player 1 deck & Player 2 deck from dropdowns → 🚀 Start Playtest. Loads instantly with all exact prints preserved.

❓ Frequently Asked Questions

Q: How often is the data updated?

A: Check the "Last Update" timestamp in the footer. Data is updated regularly as new tournaments occur.

Q: Can I use this for deck building in Pokémon TCG Live?

A: Yes! All "Copy Deck" functions output decks in Pokémon TCG Live format. Paste directly into the game.

Q: What does "international prints" mean?

A: The same card printed in different sets (e.g., Riolu MEG 76 = Riolu ASC 112). The website recognizes these automatically across all tabs.

Q: Why are Japanese cards (M3, M2a) shown?

A: Japanese sets release months before international versions. They give early insight into upcoming cards and strategies.

Q: My Rarity Switcher showed a "?" instead of ★ before — is that fixed?

A: Yes, fixed in v36. All broken symbol characters (Rarity Switcher ★, owned ✓ badge, Ace Spec ★ column) are now correctly displayed across every tab.

Q: The Playtester loaded wrong card arts from My Decks — is that fixed?

A: Yes, fixed in v35. The new direct lookup bypasses the string roundtrip, so cards with Trainer Gallery numbers (TG01), Gallery Gold (GG01), full arts, and promos all resolve to the exact saved print.

Q: What are "Combined Variants"?

A: New in v37. When the same card is printed in multiple sets (e.g. Riolu in MEG and ASC), the website automatically combines their statistics. Usage%, average count, and deck inclusion are calculated across all prints. The Deck Builder, Auto-Complete, and Card Overview all respect this — so you see the true popularity of a card regardless of which set version players used.

Q: Why do Special Energies appear before Basic Energies?

A: This is the correct sort order (fixed in v37). Special Energies (Jet Energy, Double Turbo, etc.) sort before Basic Energies (Fire, Water, etc.) across all views — matching standard deck list conventions.

Q: What is "Max Consistency" Auto-Generate?

A: The v18 algorithm looks at every deck using a card, calculates the average count among those decks (not all decks), then rounds to the nearest integer. A 5% meta-share cutoff prevents fringe cards from being included. The result is a tournament-realistic 60-card deck.

Q: Do my decks and collection sync between desktop and mobile?

A: Yes, as long as you're signed in with the same account. Firebase syncs everything in real-time.
